# MDM Device Inventory Dashboard

Hi there! Welcome to my Mobile Device Management(MDM) Dashboard

I built this project to demonstrate a clean, responsive, and interactive UI for monitoring devices. It allows users to 
quickly view system statistics and search through the device inventory in real-time.

# Tech Stack
Frontend Framework: React (Initialised with Vite)
Styling: Tailwind CSS
State management: React (useState)

## Steps to run it locally
1. Clone the repository 
open your terminal and run:
git clone https://github.com/TariqSami1/MDM-Dashboard-Project.git

2. Make sure you have Node.js installed
then run:
npm install

3. Start the server
run:
npm run dev

4. View the webapp
Now, open your browser and navigate to the link provided in the terminal
